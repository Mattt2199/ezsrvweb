# ezsrvweb

How to download and use all files on your ubuntu server:<br>
<br>
  - Download the entire repository: git clone https://github.com/Mattt2199/ezsrvweb.git<br>
  - Make the installation script executable: sudo chmod +x ezsrvweb/install.sh<br>
  - Run the installer script: sudo ./ezsrvweb/install.sh
